This change will automatically set the `title` attribute for `Menu.Item` when the `children` is a pure string, addressing the issue of long text being folded and still visible on hover. For custom `Menu.Item` where `children` is not a pure string, the `title` attribute will not be set, avoiding the double hover problem.
