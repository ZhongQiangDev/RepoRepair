The current implementation uses a single `searchProps` prop for both panels, which means the search box configuration is the same for both sides. To fix this, we need to introduce separate props for the left and right search boxes.

Here's the proposed solution:

1. Add new props `leftSearchProps` and `rightSearchProps` to the `Transfer` component.
2. Use these props in the `TransferPanel` components for the left and right panels respectively.
