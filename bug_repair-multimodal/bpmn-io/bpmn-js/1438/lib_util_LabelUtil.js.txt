## Issue Analysis

The issue is that message element names are not being rendered in BPMN diagrams. Looking at the screenshot, we can see message elements (envelope icons) on message flows, but their names aren't displayed.

The problem is in the `LabelUtil.js` file, specifically in the `getExternalLabelMid` function. This function determines the position of external labels for different BPMN elements, but it doesn't have specific handling for message elements.

## Solution

We need to modify the `getExternalLabelMid` function to handle message elements. Messages should have their labels positioned below them, similar to other elements but with appropriate positioning.
