## Issue Analysis

The issue is that message element names are not being rendered in BPMN diagrams. Looking at the provided screenshot, we can see message elements (envelope icons) on message flows, but their names are not displayed.

The problem is in the `LabelEditingProvider.js` file, which handles how labels are displayed for different BPMN elements. Currently, there's no specific handling for message elements.

## Solution

We need to add support for message elements in the `LabelEditingProvider.prototype.getEditingBBox` method. Specifically, we need to handle elements of type 'bpmn:Message' to display their names properly.
