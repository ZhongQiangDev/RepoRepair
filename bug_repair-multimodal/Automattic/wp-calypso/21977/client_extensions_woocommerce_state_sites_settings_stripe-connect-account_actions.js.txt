## Issue Analysis

The issue states that a green success banner should be displayed after successfully creating a Stripe Connect account. Looking at the code, I can see that there are action creators for handling the success case of creating a Stripe Connect account (`createSuccess`) and completing OAuth connection (`oauthConnectSuccess`), but they don't include any notification or success message.

To fix this issue, we need to modify these success action creators to include a success notification that will display the green banner.

## Solution
