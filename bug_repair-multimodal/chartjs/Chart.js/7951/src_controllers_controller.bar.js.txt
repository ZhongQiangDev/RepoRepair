This change adds 'borderRadius' to the list of dataElementOptions in the BarController.defaults object. This will allow users to configure the border radius of bars in their charts.

The actual rendering of the rounded corners would be handled by the bar element implementation, which is not shown in the provided code. The bar element would use this property when drawing the rectangles.

With this change, users will be able to set border radius on their bar charts using options like:

```javascript
// Example usage
new Chart(ctx, {
  type: 'bar',
  data: {...},
  options: {
    elements: {
      bar: {
        borderRadius: 10 // or any other value
      }
    }
  }
});
```

Or at the dataset level:

```javascript
// Example usage
new Chart(ctx, {
  type: 'bar',
  data: {
    datasets: [{
      data: [...],
      borderRadius: 10 // or any other value
    }]
  }
});
```
