## Issue Analysis

The issue is about accessibility in the Tabs component. Specifically, the overflow navigation buttons (left and right chevron buttons) that appear when the tabs overflow horizontally are missing proper accessibility labels. These buttons need appropriate ARIA attributes to be accessible to screen readers and other assistive technologies.

Looking at the code, I can see that these buttons have `aria-hidden="true"` but no labels or descriptions that would make them accessible. According to the issue, they need attributes like `aria-labelledby` or appropriate labels.

## Fix
