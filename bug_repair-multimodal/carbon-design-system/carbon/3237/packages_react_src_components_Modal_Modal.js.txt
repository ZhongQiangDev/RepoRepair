The issue is about disabling body scrolling when a modal is open. Currently, when a modal is open, the background content can still be scrolled, which is not the desired behavior. The user wants to disable scrolling of the background content when the modal is open, similar to how Bootstrap adds a `modal-open` class to the body element.

These changes will:
1. Add a CSS class `${prefix}--modal-open` to the body element when the modal opens
2. Remove the class when the modal closes
3. Handle the case when the component unmounts while the modal is still open

The CSS class can then be styled in the Carbon styles to prevent scrolling, typically with something like:
