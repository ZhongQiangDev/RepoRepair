## Issue Analysis

The issue is that VoiceOver doesn't announce the On/Off labels for the Toggle component. According to the issue description, this is because the labels have `aria-hidden="true"` attributes, which prevents screen readers from reading them.

Looking at the code, I can see that both label spans have `aria-hidden="true"`:

```javascript
<span className={`${prefix}--toggle__text--left`} aria-hidden="true">
  {labelA}
</span>
...
<span className={`${prefix}--toggle__text--right`} aria-hidden="true">
  {labelB}
</span>
```

This is preventing VoiceOver from announcing the labels. The fix is to remove these `aria-hidden="true"` attributes or provide an alternative way for screen readers to access this information.
