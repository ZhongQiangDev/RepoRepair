The issue is requesting to add a "flush alignment" prop to the Accordion component, similar to what was done for the StructuredList component. This prop would allow users to have accordion content flush with the left edge of the page, which is particularly useful in side panel situations.

Based on the specs provided, we need to add a new boolean prop called `isFlush` to the Accordion component that, when true, will align the text flush to the left end of the line dividers.

These changes add the `isFlush` prop with a default value of `false` to the Accordion component. When set to `true`, it will add the `--accordion--flush` class to the component, which can be styled in the CSS to remove the padding-left before the text, making the content flush with the left edge as requested in the issue.
