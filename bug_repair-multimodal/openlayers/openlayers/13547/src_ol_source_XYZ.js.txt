## Issue Analysis

The issue is that the `gutter` option is currently only supported by the TileWMS source, but it should be available to more tile sources like XYZ and DataTile. The gutter (or buffer) is used to prevent tile edge effects when interpolating between tiles.

Looking at the XYZ.js file, I can see that the `gutter` option is not included in the Options typedef or passed to the TileImage constructor in the XYZ class.

## Solution

1. Add the `gutter` option to the Options typedef in XYZ.js
2. Pass the gutter option to the TileImage constructor
